# CyberShield 版本记录

## v2026.06.06 - 核心功能完善

### 新增功能
- ✅ 自定义屏蔽词管理界面（添加、删除、清空全部）
- ✅ 屏蔽规则可视化查看（硬关键词、软关键词、正则表达式）
- ✅ 多语言词库扩展（28种语言支持）
- ✅ 「关于」页面 + 隐私声明
- ✅ 面板三页布局（控制、日志、关于）

### 平台适配增强
- ✅ YouTube：新增 `getCurrentUser()` + DOM 模拟拉黑
- ✅ Reddit：新增 `getCurrentUser()` + API 拉黑 + DOM 降级
- ✅ 微博：新增 `getCurrentUser()` + API 拉黑 + DOM 降级
- ✅ B站：完整支持（评论 + 私信扫描 + API 拉黑）
- ✅ Twitter/X：DOM 模拟拉黑支持

### 修复优化
- ✅ 自定义关键词删除确认提示
- ✅ 重复拉黑检测（同一用户只拉黑一次）
- ✅ 取证截图功能接入
- ✅ MutationObserver 节流优化
